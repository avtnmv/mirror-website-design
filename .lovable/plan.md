

# Poslední Tečka — Копия сайта

Одностраничный сайт финансово-юридической компании по возврату потерянных инвестиций. Чешский язык, современный корпоративный дизайн.

## Дизайн-система
- **Цвета**: Тёмно-синий (#0A1F44) как основной, красный/малиновый (#E63946) для акцентов и CTA-кнопок, белый фон для секций контента
- **Шрифт**: Современный sans-serif (Inter), крупные заголовки, чистая типографика
- **Стиль**: Минималистичный корпоративный, много белого пространства, иконки с цветными кружками

## Структура страницы (сверху вниз)

### 1. Шапка (Header)
- Логотип «Poslední tečka» слева
- Навигация: «Jak vám pomůžeme», «O nás», «Kontakty»
- CTA-кнопка «Chci vědět více» справа

### 2. Герой-секция
- Тёмно-синий фон с заголовком «Zachráníme vám i zdánlivě ztracené investice»
- Подзаголовок о помощи в возврате средств от мошеннических брокеров
- Красная CTA-кнопка «Chci vaši pomoc»
- Значки доверия (лет опыта, довольные клиенты)
- Изображение человека справа (с синим/красным освещением)

### 3. Секция услуг
- Заголовок «Vrátíme vám investice do akcií i kryptoměn»
- Два столбца: «Co neřešíme» (что не делаем) и «S čím vám pomůžeme» (чем помогаем)
- Иконки с описаниями для каждого пункта
- CTA-кнопка внизу

### 4. Секция «Kolik peněz zachráníme»
- Статистика по сумме возвращённых средств
- Галочки с достижениями компании

### 5. Секция процесса сотрудничества
- Заголовок «Jak vypadá naše spolupráce»
- 3 шага в карточках с иконками: Контакт → Проверка → Подписание договора
- CTA-кнопка

### 6. Секция ценообразования
- Заголовок «Hrajeme fér!»
- Таблица/сетка с ценовыми диапазонами (до 499 999 Kč, до 999 999 Kč, и т.д.)
- Указание цен с/без DPH

### 7. CTA-баннер
- Тёмно-синий фон, заголовок «Jsme na vaší straně»
- Белая кнопка

### 8. Секция отзывов
- Заголовок «Co říkají naši klienti»
- Карточки с отзывами клиентов

### 9. Футер
- 3 колонки: информация о компании, меню, контактная информация
- Логотип «Poslední tečka»

## Технические детали
- Одностраничное React-приложение, без бэкенда
- Адаптивная вёрстка (десктоп + мобайл)
- Плавная прокрутка к секциям через навигацию
- Все тексты на чешском языке по образцу скриншота
